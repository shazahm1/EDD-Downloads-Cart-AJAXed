Ajaxed version of the cart widget for eshop (wp-super-cache enabled).

== Description ==

This plugin is just an ajaxed version of the widget cart for eshop. In this way, you can use wp-super-cache (and probably others cache system... let me know).

== Installation ==

1. Install it :)
2. Activate it.
3. Go to the widget panel and add it to your sidebar.
4. Here it is !
